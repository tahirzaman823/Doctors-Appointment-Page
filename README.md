# 🏥 Doctor Appointment Booking App

A clean, multi-step doctor appointment booking UI built with vanilla HTML, CSS & JavaScript — no frameworks, no dependencies.

## 💡 What Is This?

A ready-to-use appointment booking interface for any clinic, hospital, or healthcare setup. Patients can select a specialist, enter their details, and receive a token number — all in a simple 3-step flow.
